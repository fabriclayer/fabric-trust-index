# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in the Fabric Trust Index, please report it responsibly.

**Email:** info@motherbird.ai

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial assessment:** Within 5 business days
- **Resolution target:** Within 30 days for critical issues

## Scope

This policy covers:
- The Trust Index web application (trust.fabriclayer.ai)
- The scoring and collection pipeline
- API endpoints
- Database access controls

## Out of Scope

- Third-party services we monitor (report those to the respective maintainers)
- Trust scores or ratings disputes (use the Report Issue feature on the site)

## Disclosure Policy

We follow coordinated disclosure. We ask that you do not publicly disclose the vulnerability until we have had a reasonable opportunity to address it. We will credit reporters in our release notes unless anonymity is requested.
